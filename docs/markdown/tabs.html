linkReferencing Tabs in Code

You can reference another tab from a code snippet within one tab:

1> :Tabs

2> > :Tab title=Component Code

3> >

4> > ```tsx | index.tsx

5> > import { CompStyles } from './styles'  // @see tab:Styling Code

6> >

7> > export function Comp() {

8> >   ... 

9> > }

10> > ```

11>

12> > :Tab title=Styling Code

13> > ```tsx | style.ts

14> > export const CompStyle = { ... }

15> > ```

index.ts
1import { CompStyles } from './styles'  // @see tab:Styling Code

2

3export function Comp() {

4  ...

5}

style.ts
