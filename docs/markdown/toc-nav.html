linkToC Navigation

The ToCPrevNext component scans through the ToC, matches the URL of current page and extrapolates the previous and next pages, showing beautiful buttons for navigating to them:

1> :ToCPrevNext



linkCustom Labels

You can customize the labels of the buttons via prev-label and next-label attributes:

1> :ToCPrevNext prev-label=...Previously, next-label=Up Next...



linkCustom Icons

You can customize the icons of the buttons via prev-icon and next-icon attributes:

1> :ToCPrevNext prev-label=...Previously, prev-icon=arrow_back, next-label=Up Next..., next-icon=arrow_forward

The icons are read from the configured icon-font, which is by default Material Icons. You can override this in .codedoc/config.ts:

.codedoc/config.ts
1// ...

2export const config = /*#__PURE__*/configuration({

3  // ...

4  page: {

5    // ...

6    fonts: {

7      // ...

8      icon: {

9        name: 'My-Icon-Font',                                      // --> name of the icon-font

10        url: 'https://some-where-on-the.cloud/fonts/my-icon-font', // --> the URL of the icon-font

11      }

12    },

13    // ...

14  },

15  // ...

16});
