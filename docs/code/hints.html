linkCode Hints

If a line in a code-snippet has the syntax --> hint text, then the hint text will be displayed on a hint-box when the line is hovered (the cursor is upon it):

1```cpp

2void func() {        // --> this function does nothing

3}

4```

1```js

2function func() {     /* --> this function does nothing */

3}

4```

1```python

2def func():           # --> this function does nothing

3  pass

4```

1```html

2<div hidden></div>    <!--> this displays nothing -->

3```

1void func() {        // --> this function does nothing

2}

1function func() {     /* --> this function does nothing */

2}

1def func():           # --> this function does nothing

2  pass

1<div hidden></div>    <!--> this displays nothing -->

linkLanguage Support

This feature is currently only supported in languages where one of the following comment syntaxes is valid:

1// single-line comment

1/* multi line comment */

1# comment starting with a hash

1<!-- HTML style comment -->
